HTML

  • Style element defines the CSS Styles
  • Base element defines the base URL
  • Link element connect an external resource to the HTML document
  • Meta defines metadata, info about the page used by the browser
  • Script element embeds client-side scripts in HTML document
  • Charset means character set
  • HTTP-equiv element sets an HTTP header for content
  • Viewport element sets default view of a webpage
  • Section elements are semantic elements that define a section in a document that contains thematic content
  • Javascript uses "id" to identify elements and select them
  • "h2" indicates a second tier of headings
  • "ul" creates an unordered list of items
  • inside "ul" we need to wrap each list iten in an "li" element

CSS

  • 3 Types of CSS: Inline, Internal and External
  • "Separation of Concerns" refers to separating code into differnt files based on use
  • You can combine two Selectors with a comma
  • "img" selector assigns property values to any image in the markup
  • "link" element creates the connection to the external file
  • "rel" attribute specifies the relationship between the current document and the linked document resource
  • "href" attribute specifies the location of the external resource
  • The period in ".card" designates htis is a class selector. ANy HTML element assigned to the class "card" will gain these properties
  • "box-shadow" CSS property adds shadow effects around an element's frame, on an X,Y axis
